Using this Hub

This site hosts materials for the PR-for-AATD co-design project. You don’t need any technical skills—just click the links below to view or download each item.

Navigation

  • Home – project overview.
  • About – aims and background.
  • Prototypes – download forms and booklets.
  • Workshops – agendas, case studies, and feedback forms.
  • Outputs – reports and visual abstract.

Tips

  • If a link opens a PDF, use your browser’s download button to save it.
  • If you see a “404” page, the file is still being updated—check again later.
  • You can open this site on any phone, tablet, or computer.
